## ✨ ¿Qué hace este proyecto?

Tasker permite gestionar formularios dinámicos, responder asignaciones y visualizar reportes con evidencia multimedia.

### Funcionalidades destacadas

- ✅ Formularios dinámicos con múltiples tipos de campo.
- 📷 Captura de imagen e identificación (ID) desde cámara.
- ✍️ Firma digital en campo.
- 📎 Carga de documentos y evidencias.
- 📍 Geolocalización (manual y por alcance/scope).
- 📊 Visualización de reportes, respuestas y evidencias.
- 🔎 Escaneo de códigos de barras con ZXing.
- 🔐 Control de permisos por usuario/rol.
